# PC Builder // MVP

Минималистичный текстовый конструктор для подбора комплектующих ПК.

## ️ Что это такое

Веб-приложение для сборки компьютера с проверкой совместимости компонентов. Без изображений, только текст и характеристики.

## ✨ Возможности

- **Ручной подбор** — выбирайте каждый компонент из каталога
- **Проверка совместимости** — автоматическая проверка сокета, типа RAM, габаритов, TDP
- **Готовые пресеты** — 8 сборок от офисной до ультра-4K
- **Сохранение сборок** — храните варианты локально в браузере
- **Экспорт** — копируйте список компонентов в буфер обмена
- **Тёмная/светлая тема** — переключение в один клик
- **Поиск** — быстрый поиск компонентов по названию
- **Мобильная адаптация** — работает на телефонах

##  Быстрый старт

1. Откройте сайт: [https://fatalko.github.io/pc-builder/](https://fatalko.github.io/pc-builder/)
2. Выберите пресет или соберите ПК вручную
3. Сохраните сборку или скопируйте список
