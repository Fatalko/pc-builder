# PC Builder // MVP

Минималистичный текстовый конструктор для подбора комплектующих ПК с проверкой совместимости.

## ️ Что это такое

Веб-приложение для сборки компьютера без изображений — только текст и характеристики. Автоматически проверяет совместимость компонентов и блокирует несовместимые варианты.

**Сайт:** [https://fatalko.github.io/pc-builder/](https://fatalko.github.io/pc-builder/)

## ✨ Возможности

### 🎯 Подбор комплектующих
- **Ручной выбор** — каталог из 100+ актуальных комплектующих
- **Проверка совместимости в реальном времени** — автоматическая фильтрация несовместимых компонентов
- **Умная фильтрация** — несовместимые варианты становятся неактивными с пометкой
- **Поиск по каталогу** — быстрый поиск компонентов по названию

### 💾 Работа со сборками
- **Сохранение сборок** — хранение в localStorage браузера
- **История сборок** — просмотр всех сохранённых вариантов
- **Загрузка сборки** — клик по карточке восстанавливает сборку в конструкторе
- **Удаление сборок** — управление историей
- **Экспорт в текст** — копирование списка компонентов в буфер обмена

### ✅ Проверка совместимости
Система автоматически проверяет и блокирует несовместимые комбинации:
- **Сокет CPU ↔ Материнская плата** (LGA1700, AM4, AM5)
- **Тип RAM ↔ Материнская плата** (DDR4, DDR5)
- **Длина видеокарты ↔ Корпус** (в миллиметрах)
- **Высота кулера ↔ Корпус** (в миллиметрах)
- **Поддержка сокета кулером** (LGA1700, AM4, AM5)
- **Мощность БП ↔ TDP системы** (с запасом 20%)

## 🚀 Быстрый старт

1. Откройте сайт: [https://fatalko.github.io/pc-builder/](https://fatalko.github.io/pc-builder/)

4. Сохраните сборку или скопируйте список компонентов
